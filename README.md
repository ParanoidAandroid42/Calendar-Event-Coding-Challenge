
## Project setup

`npm i`

## Project run
`npm run dev`

## Test
Use 'layOutDay(calendarEvents: [])' function to test. As calendarEvents parameter give which event inputs you want as array. Event type: {start: number, end: number }

## Example data to test
   <pre>
       layOutDay([{ start: 30, end: 150 },
                  { start: 540, end: 600 },
                  { start: 600, end: 720 },
                  { start: 580, end: 720 },
                  { start: 560, end: 620 },
                  { start: 610, end: 670 }])
    </pre>
